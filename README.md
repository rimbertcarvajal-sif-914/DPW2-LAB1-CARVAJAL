## 🛠️ Actividades del Laboratorio

### 1. Sistema de Rejillas (Grid System)
- Maquetar la estructura de un **Dashboard Administrativo**.
- Debe incluir: Sidebar (barra lateral) y área de contenido principal.
- Uso obligatorio de clases de Bootstrap: `.container-fluid`, `.row`, `.col-*`.

### 2. Media Queries y Personalización
- Crear una carpeta `css/` y un archivo `style.css`.
- Implementar al menos 3 puntos de ruptura (breakpoints) para ajustar colores y tamaños de fuente.
- Asegurar que el Sidebar se oculte automáticamente en dispositivos móviles.

### 3. Componentes y Formularios
- Implementar una `Navbar` con menú desplegable.
- Crear un formulario de **Registro de Usuarios** con validación visual (`.is-valid`).
- Añadir botones de acción con diferentes estados.
